# Specification

## Summary
**Goal:** Build a birthday wish sharing app where users create personalized birthday greetings with an interactive candle-blowing experience.

**Planned changes:**
- Create a sender form page to input recipient name and custom birthday message
- Generate unique shareable links that encode the greeting data
- Store greeting data in the backend associated with unique identifiers
- Create a recipient view displaying a birthday cake with lit candles centered on the cake
- Display "Blow the candles!" prompt on the recipient view
- Implement microphone-based candle-blowing interaction that detects blowing sounds
- Show personalized happy birthday message with recipient name and sender's custom message after candles are blown out
- Design festive visual theme with warm, cheerful colors (avoiding blue and purple), playful typography, and celebratory elements like balloons and confetti

**User-visible outcome:** Users can create personalized birthday greetings by entering a name and message, receive a shareable link, and recipients can visit the link to interact with a birthday cake by blowing out the candles through their microphone, revealing the custom birthday message.
