# Specification

## Summary
**Goal:** Match candle colors and styling to the birthday cake's decoration theme for visual cohesion.

**Planned changes:**
- Update candle rendering in BirthdayCake component to dynamically style candle colors that complement the cake's color scheme
- Adjust candle appearance (shape, size, decorative elements) to match the cake's decorative theme
- Ensure visual consistency between lit and extinguished candle states
- Harmonize flame colors with the candle and cake color palette

**User-visible outcome:** Candles will visually harmonize with the cake decoration, creating a cohesive and aesthetically pleasing birthday cake display.
